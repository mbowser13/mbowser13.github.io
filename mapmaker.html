Python Map Maker Script


See below for summary or go to my GitHub page to view the code

At Wetland Studies and Solutions, Inc. our GIS staff work closely with the wetland scientists to help create map packages for their delineation reports that are submitted to the United States Army Corps of Engineers. These map packages consist of 8 maps. The maps included in the package contain the following information; location information, USGS quads, NWI wetland data,soil data, FEMA floodplain data, aerials, and drought maps. The goal of this program is to automate the creation of all of these maps.

Goals

1) Use the python URL Module to navigate to two different web pages and download and save various images relating to drought conditions based on a user inputed date

2) Insert downloaded drought maps into mxd templates

3) Insert user supplied shapefile into all delineation report exhibits (8 maps)

4) Update project name text element in maps based on user inputted name

5) Save the mxds to user specified file location

6) Export mxds to pdf “map book”
